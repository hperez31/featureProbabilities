Efficient Computation of Feature Probabilities in Variability Models

Download this project as a .zip file Download this project as a tar.gz file

Context.

Variability models play a key role in product line engineering as they are used to represent the common and variable features that products may include, and what constraints among the features must be satisfied to guarantee the validity of the products. Valuable analysis operations on variability models can be performed by black box reusing logic engines, such as SAT-solvers and binary decision diagram libraries. Unfortunately, such kind of reuse implies long computation times for operations that need calling the engines many times. To overcome this problem, we work on new algorithms that directly deal with the data structure of a binary decision diagram encoding a variability model. In particular, the algorithm published in this site is specifically designed to compute feature commonalities.

Description

This repository provides an algorithm to compute feature commonalities, which is written in the C++ programming language as an extension of the Buddy BDD library.

How to Install BuDDy.

Follow the instructions in this section to install the BuDDy BDD package on your system, if it is not already installed.

  • Download buddy-2.4.tar.gz (766,5 kB).
  • Unzip "gunzip buddy-2.4.tar.gz".
  • Untar "tar xvf buddy-2.4.tar".
  • "./configure"
  • "make"
  • "make install"

BuDDy Documentation

The buddy-2.4/doc directory contains documentation for the BuDDy package in postscript. An introduction to binary decision diagrams is described in the bddnotes.ps file. In addition, you can find a full description of the buddy package in buddy.ps.
